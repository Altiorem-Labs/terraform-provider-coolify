# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-01-27

### Added

* Initial public release of the Coolify Terraform Provider
* Provider scaffolding and core setup
* `coolify_project` resource for managing Coolify projects
* Environment management support per project with `coolify_environment`
* Initial GoReleaser configuration for multi-platform builds

### Notes

* This is an early release.
* Breaking changes may occur before v1.0.0.
* Feedback and contributions are welcome.
